# Security Policy

## Supported versions

tickflow is pre-1.0; only the latest released version receives fixes.

| Version | Supported |
| --- | --- |
| 0.3.x | ✅ |
| < 0.3 | ❌ |

## Reporting a vulnerability

Please report security issues privately using GitHub's
[private vulnerability reporting](https://github.com/davidy749/tickflow/security/advisories/new)
rather than opening a public issue.

tickflow is a numerical library with no network or file-execution surface beyond
reading CSVs you point it at, so the realistic risk is limited — but reports of
unsafe parsing or dependency issues are still appreciated. Expect an initial
response within a week.
